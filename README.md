# ANDORPEL-CASHFLOW-PROYECCION

## Clonación local para modificaciones

Para trabajar en cambios sin tocar este repositorio base, se puede crear una copia local así:

```bash
git clone /workspace/ANDORPEL-CASHFLOW-PROYECCION /workspace/ANDORPEL-CASHFLOW-PROYECCION-clone
```

En esta sesión ya se creó el clon en:

- `/workspace/ANDORPEL-CASHFLOW-PROYECCION-clone`
